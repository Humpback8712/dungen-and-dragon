# dungen-and-dragon

这是一个逻辑思路基于DND的一个游戏，我打算先做一个简单版本用于作业，如果未来有时间我会将其完善，毕竟DND规则及多，如果有机会我会将其做成多人游戏放在我的网站上，做一个线上聊天室，就可以进行多人游戏了。

DND的游戏思路跟普通的游戏不太一样，这是一个充满想象力的游戏，所以程序能做到的只是将规则翻译为计算机语言，而真正的核心在于DM，所以说难一点都不难，但是由于DND庞大的设定（500 多页的玩家手册），我不知道什么时候才能完成，所以欢迎看到的朋友们，如果有兴趣，可以folk我，或者加我的微信，我们一起将这个神奇的世界建立起来。
